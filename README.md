# Polygon Beat Slicer

Browser-based audio-reactive animation tool. Polygons are sliced in sync with audio, creating a persistent composition of fragmented shapes.

**Live:** [kinetic-animation.pages.dev](https://kinetic-animation.pages.dev)

## Features

- **Beat Detection** — bass-energy threshold via Web Audio API analyser
- **Cue Sheet** — every beat at 130 BPM
- **Built-in synth** — auto-generates a 30s electronic beat track
- **Infinite loop** — toggle for seamless playback
- **Recursive slicing** — every 4 cuts, the camera zooms into the colored piece

## Dev

```bash
npm install
npm run dev
npm run build
```
